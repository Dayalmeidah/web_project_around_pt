# Tripleten web_project_around_pt
🚀 Sobre o Projeto:
Este projeto foi desenvolvido como parte do curso de Desenvolvimento Web. O foco principal foi a criação de uma interface dinâmica utilizando HTML5, CSS3 (Metodologia BEM) e JavaScript assíncrono, garantindo que a experiência do usuário seja fluida e responsiva.

🧐 Visão Geral:
Ao executar a tarefa de criaçao deste projeto, aprendi como usar o JavaScript e como deixar minha pagina interativa e dinamica. Sendo assim, o projeto inclui a funçao de adicionar novos cartoes, a opçao de curtir e deletar os mesmos, inclui tambem a mudança no nome da profissao do personagem sitado no projeto.

🚀 Tecnologias:
Este projeto foi desenvolvido utilizando as seguintes tecnologias:

HTML5
CSS3 (Flexbox, Grid,BEM)
JavaScript (ES6+)

✨ Funcionalidades:

Validação Dinâmica
Botão Fixo
Acessibilidade
Clean Code
