# Tripleten web_project_around_pt
🚀 Sobre o Projeto
Este projeto foi desenvolvido como parte do curso de Desenvolvimento Web da TripleTen. O foco principal foi a criação de uma interface dinâmica utilizando HTML5, CSS3 (Metodologia BEM) e JavaScript assíncrono, garantindo que a experiência do usuário seja fluida e responsiva.

** Funcionalidades Implementadas:

Edição de Perfil: O usuário pode alterar seu nome e descrição através de um modal dinâmico.

Manipulação do DOM: Os dados do perfil são validados e atualizados na página sem a necessidade de recarregamento.

Galeria de Cartões: Exibição dinâmica de fotos e títulos de locais turísticos.

Modais Interativos: Sistema de abertura e fechamento de pop-ups com foco em funções reutilizáveis.


🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica.

CSS3: Estilização avançada com uso de Flexbox e Grid.

Metodologia BEM: Organização de classes para um código CSS escalável e limpo.

JavaScript (ES6+):

Manipulação de eventos (addEventListener).

Gerenciamento de estados de modais.

Tratamento de submissão de formulários (preventDefault).

📂 Estrutura do Código
O código foi organizado seguindo o princípio de responsabilidade única:

openModal() / closeModal(): Funções genéricas para controle de visibilidade.

handleProfileFormSubmit(): Gerencia a lógica de atualização de dados e prevenção de comportamento padrão do navegador.

fillProfileForm(): Garante que os campos do formulário sempre abram com os dados atuais da página.
